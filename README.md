# LinearLayOut
用线性布局生成一个4*4的表格，需要四块LinearLayout，每块LinearLayout包含4个textView，横向布局，设置marginRight的值进行间隔，然后在最外面包含一个总的线性布局框，垂直布局，设置marginBottom值
